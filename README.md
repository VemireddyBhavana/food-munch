### Prerequisites

Before you begin, ensure you have met the following requirements:

* [Git](https://git-scm.com/downloads "Download Git") must be installed on your operating system.
* [Node.js](https://nodejs.org/en/ "Download Node.js") must be installed.

### Run Locally

To run **Food Munch** locally, run these commands in your terminal:

```bash
git clone https://github.com/Bhavana80770/food-munch.git
cd food-munch
npm install
npm run dev
```
